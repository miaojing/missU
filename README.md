# 交通
* 做黄色公交：950每人 
* 地铁 900 主要以地铁出行为主：首尔地铁运行时间为凌晨5点至夜间1点
* taxi: 2km以内2400韩元 超过2km以后每144m加收100韩元。以低于时速15km的速度行驶时，每35秒加收100韩元。00 :00~04 :00之间乘车时需多支付20%的深夜附加费。
* 观光巴士：市區循環路線 10,000（约53人民币）
* T-money 公交卡  到处可以刷卡 必须一人一张 已经淘宝上买好
